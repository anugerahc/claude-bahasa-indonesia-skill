# Claude Bahasa Indonesia Skill (MCP Plugin)

Plugin berbasis Model Context Protocol (MCP) untuk memaksa Claude (di VS Code Extension / Desktop) agar selalu merespons menggunakan Bahasa Indonesia yang ringkas, jelas, dan kasual (*lu-gue*).

## Fitur
- Otomatis mengubah preferensi bahasa agen ke Bahasa Indonesia.
- Tetap mempertahankan istilah teknis dan *error log* dalam bahasa asli (English).

## Cara Pasang
1. Buka menu **Manage Plugins / MCP Servers** di ekstensi VS Code Anda.
2. Masukkan URL repositori ini:
   `https://github.com/anugerahc/claude-bahasa-indonesia-skill`
3. Klik **Add**.
